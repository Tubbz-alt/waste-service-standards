Data models

This gives an overview of the data models:

Model Description
Waste service A waste or recycling service provided by the council for citizens.
Round plan A truck collecting one or more material streams from a set of containers on a repeating schedule.
Round An instance of a round plan that has been executed on a particular date.
Waste event An issue or check that is recorded during a round.
Container Where materials are stored for collection.
Material stream The types of materials that can be collected.
